    Convertible Bond

    For Taiwan stock convertible bonds, we have 5 datasets, as listed below:

    • Convertible Bond Overview TaiwanStockConvertibleBondInfo
    • Convertible Bond Daily Trading Information TaiwanStockConvertibleBondDaily
    • Convertible Bond Institutional Investors Daily Trading TaiwanStockConvertibleBondInstitutionalInvestors
    • Convertible Bond Daily Overview TaiwanStockConvertibleBondDailyOverview
    • Convertible Bond Monthly Analysis TaiwanStockConvertibleBondMonthlyAnalysis

    Convertible Bond Overview TaiwanStockConvertibleBondInfo (only available to backer/sponsor members)¶

    Example

    from FinMind.data import DataLoader

api = DataLoader()
# api.login_by_token(api_token='token')
df = api.taiwan_stock_convertible_bond_info()

    import requests
import pandas as pd
url = "https://api.finmindtrade.com/api/v4/data"
token = "" # See login section to obtain a token
headers = {"Authorization": f"Bearer {token}"}
parameter = {
    "dataset": "TaiwanStockConvertibleBondInfo",
}
data = requests.get(url, headers=headers, params=parameter)
data = data.json()
data = pd.DataFrame(data['data'])
print(data.head())

    library(httr)
library(data.table)
library(dplyr)
token = "" # See login section to obtain a token
url = 'https://api.finmindtrade.com/api/v4/data'
response = httr::GET(
    url = url,
    query = list(
        dataset="TaiwanStockConvertibleBondInfo"
    ),
    add_headers(Authorization = paste("Bearer", token))
)
data = response %>% content
df = do.call('cbind',data$data) %>%data.table
head(df)
