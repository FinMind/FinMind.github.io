Methods

FinDataList

FinDataList(dataset)

Load Data List from dataset.

Arguments

  • dataset : String.

    • InstitutionalInvestors : Taiwan Stock Institutional Investors Buy and Sell.
    • CrudeOilPrices : Crude Oil Prices.
    • ExchangeRate : Exchange Rate.
    • InterestRate : Interest Rate.
    • GovernmentBonds : Government bond.
    • EnergyFuturesPrices : Energy Futures Prices.

    The detail can follow Data Introduction.

Return : List.

FinData

FinData(dataset,select,date)

Load Data from dataset.

Arguments

  • dataset : String.

    • TaiwanStockInfo : Taiwan Stock Information.
    • USStockInfo : US Stock Information.
    • UKStockInfo : UK Stock Information.
    • JapanStockInfo : Japan Stock Information.
    • EuropeStockInfo : Europe Stock Information.
    • TaiwanStockPrice : Taiwan Stock Price.
    • USStockPrice : US Stock Price.
    • UKStockPrice : UK Stock Price.
    • JapanStockPrice : Japan Stock Price.
    • EuropeStockPrice : Europe Stock Price.
    • FinancialStatements : Taiwan Stock & US Stock Financial Statements.
    • TaiwanStockStockDividend : Taiwan Stock Stock Dividend.
    • InstitutionalInvestorsBuySell : Taiwan Stock Institutional Investors Buy and Sell.
    • TaiwanStockMarginPurchaseShortSale : Taiwan Stock Margin Purchase Short Sale.
    • CrudeOilPrices : Crude Oil Prices.
    • ExchangeRate : Exchange Rate. ( contains G7 )
    • InterestRate : Interest Rate. ( contains G7 )
    • GovernmentBonds : Government bond. ( contains G7 )
    • EnergyFuturesPrices : Energy Futures Prices.
    • GoldPrice : Gold Price.
    • BalanceSheet : Taiwan Stock Balance Sheet.
    • HoldingSharesPer : Taiwan Stock Holding Shares Per.
    • Shareholding : Taiwan Stock Shareholding.
    • RawMaterialFuturesPrices : Raw Material Futures Prices. ( meats, grains, energies, softs, metals)

    The detail can follow Data Introduction.

  • select : String or List.

    • Case 1 : No need input.
      • TaiwanStockInfo, USStockInfo, UKStockInfo, JapanStockInfo, EuropeStockInfo.
    • Case 2 : Input stock id from TaiwanStockInfo, USStockInfo, UKStockInfo, JapanStockInfo, EuropeStockInfo.
      • TaiwanStockPrice, USStockPrice, UKStockPrice, JapanStockPrice, EuropeStockPrice, FinancialStatements, TaiwanStockStockDividend, InstitutionalInvestorsBuySell, TaiwanStockMarginPurchaseShortSale.
    • Case 3 : Input element from function FinDataList.
      • InstitutionalInvestors, CrudeOilPrices, ExchangeRate, InterestRate, GovernmentBonds, EnergyFuturesPrices.

  • date : String.

    • Input type : 'yyyy-mm-dd' ( e.g : '2017-03-05' )

Return : DataFrame.

Data Introduction

StockInfo

  • TaiwanStockInfo : Column stock_id, stock_name, kinds.

  • USStockInfo : Column stock_id, stock_name, Country, IPOYear, Country, MarketCap, Subsector.

  • JapanStockInfo : Column stock_id, stock_name, Sector, Exchange.

  • UKStockInfo : Column stock_id, stock_name, Country.

  • EuropeStockInfo : Column stock_id, stock_name, Market.

StockPrice

Column date, Volume, Open, High, Low, Close, Adj_Close.

FinancialStatements

Column date,stock_id,type,value.

TaiwanStock Other Information

  • Taiwan Stock Stock Dividend : Column : date, stock_id, Retained_Earnings, Capital_Reserve, Ex_right_trading_day, total_employee_bonus_stock_shares, Cash_dividend, Ex_dividend_transaction_day, total_employee_bonus_shares, Directors_remuneration.

  • Institutional Investors : Institutional Investors buy or sell. ( Dealer, Dealer Hedging, Foreign Investor, Investment Trust )
    Column : name, buy, sell, date.

  • Taiwan Stock Margin Purchase Short Sale : Column : stock_id, stock_name, MarginPurchaseBuy, MarginPurchaseSell, MarginPurchaseCashRepayment, MarginPurchaseYesterdayBalance, MarginPurchaseTodayBalance, MarginPurchaseLimit, ShortSaleBuy, ShortSaleSell, ShortSaleCashRepayment, ShortSaleYesterdayBalance, ShortSaleTodayBalance, ShortSaleLimit, OffsetLoanAndShort, Note, date.

  • Taiwan Stock Institutional Investors Buy and Sell : Column : Chinese_Investor,Foreign_Investor,Investment_Trust,Dealer_self, Dealer_Hedging.
  • Balance Sheet : Column date,stock_id,type,value.

  • Holding Shares Per : Column : HoldingSharesLevel,people,unit,percent, stock_id,date.
  • Shareholding : Column : stock_id,stock_name,InternationalCode,NumberOfSharesIssued, ForeignInvestmentRemainingShares,ForeignInvestmentShares, ForeignInvestmentUpperLimitRatio,ChineseInvestmentUpperLimitRatio, ,RecentlyDeclareDate,date.

International Financial Data

  • Crude Oil Prices : Column : price, date, name.
  • Exchange Rate : Column : InterbankRate, InverseInterbankRate, date, country.
  • Interest Rate : Column : date, interest_rate, country, full_country_name.
  • Government Bonds : Column : date, Price, Open, High, Low, ChangePercent, bound_type.
  • Energy Futures Prices : Column : date, Price, Open, High, Low, Vol, ChangePercent, futures_type.
  • Gold Price : Column : date, Price.
  • Raw Material Futures Prices : Column : date, Price, Open, High,Low, Vol, ChangePercent, futures_type.
