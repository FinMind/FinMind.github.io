Methods

FinDataList

FinDataList(dataset)

Load Data List from dataset.

Arguments

  • dataset : String.

    • StockPrice : Taiwan Stock Price.
    • FinancialStatements : Taiwan Stock Financial Statements.
    • StockDividend : Taiwan Stock Stock Dividend.
    • InstitutionalInvestors : Taiwan Stock Institutional Investors Buy and Sell.
    • CrudeOilPrices : Crude Oil Prices.
    • ExchangeRate : Exchange Rate.
    • InterestRate : Interest Rate.
    • GovernmentBonds : Government bond.
    • EnergyFuturesPrices : Energy Futures Prices.

    The detail can follow Data Introduction.

Return : List.

FinData

FinData(dataset,select,date)

Load Data from dataset.

Arguments

  • dataset : String.

    • StockInfo : Taiwan Stock Information.
    • StockPrice : Taiwan Stock Price.
    • FinancialStatements : Taiwan Stock Financial Statements.
    • StockDividend : Taiwan Stock Stock Dividend.
    • InstitutionalInvestors : Taiwan Stock Institutional Investors Buy and Sell.
    • CrudeOilPrices : Crude Oil Prices.
    • ExchangeRate : Exchange Rate.
    • InterestRate : Interest Rate.
    • GovernmentBonds : Government bond.
    • EnergyFuturesPrices : Energy Futures Prices.
    • GoldPrice : Gold Price.

    The detail can follow Data Introduction.

  • select : String or List. Only been input element from FinDataList.

  • date : String. Input type : 'year-month-days' ( e.g : '2017-03-05' )

Return : DataFrame.

Data Introduction

  • StockInfo : More than 1000 stock.

  • StockPrice : All of stock price of stockinfo. The date started in 1992.
    Cotain date, Trading_Volume, Trading_money, open, max, min, close, spread, Trading_turnover.

    • REV : Gross Revenue
    • COST : Cost of Goods Sold or Manufacturing
    • PRO : Gross Profit
    • GM : Gross margin

    • OI : Operating Income
    • OIM( Operating Profit Margin
    • BTAX( Income before Tax
    • BTAXM( Income before Tax Margin

    • NI( Net Income

    • EPS( Earnings Per Share
