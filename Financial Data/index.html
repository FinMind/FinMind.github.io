Methods

FinDataList

FinDataList(dataset)

Load Data List from dataset.

Arguments

  • dataset : String.

    • InstitutionalInvestors : Taiwan Stock Institutional Investors Buy and Sell.
    • CrudeOilPrices : Crude Oil Prices.
    • ExchangeRate : Exchange Rate.
    • InterestRate : Interest Rate.
    • GovernmentBonds : Government bond.
    • EnergyFuturesPrices : Energy Futures Prices.

    The detail can follow Data Introduction.

Return : List.

FinData

FinData(dataset,select,date)

Load Data from dataset.

Arguments

  • dataset : String.

    • TaiwanStockInfo : Taiwan Stock Information.
    • TaiwanStockPrice : Taiwan Stock Price.
    • TaiwanStockFinancialStatements : Taiwan Stock Financial Statements.
    • TaiwanStockStockDividend : Taiwan Stock Stock Dividend.
    • InstitutionalInvestors : Taiwan Stock Institutional Investors Buy and Sell.
    • USStockInfo : US Stock Information.
    • USStockPrice : US Stock Price.
    • JapanStockInfo : Japan Stock Information.
    • JapanStockPrice : Japan Stock Price.
    • CrudeOilPrices : Crude Oil Prices.
    • ExchangeRate : Exchange Rate. ( contains G7 )
    • InterestRate : Interest Rate. ( contains G7 )
    • GovernmentBonds : Government bond. ( contains G7 )
    • EnergyFuturesPrices : Energy Futures Prices.
    • GoldPrice : Gold Price.

    The detail can follow Data Introduction.

  • select : String or List.

    • Case 1 : No need input.
      • TaiwanStockInfo, USStockInfo, JapanStockInfo.
    • Case 2 : Input stock id from TaiwanStockInfo or USStockInfo or JapanStockInfo.
      • TaiwanStockPrice, TaiwanStockFinancialStatements, TaiwanStockStockDividend, USStockPrice, JapanStockPrice.
    • Case 3 : Input element from FinDataList.
      • InstitutionalInvestors, CrudeOilPrices, ExchangeRate, InterestRate, GovernmentBonds, EnergyFuturesPrices.

  • date : String.

    • Input type : 'yyyy-mm-dd' ( e.g : '2017-03-05' )

Return : DataFrame.

Data Introduction

Taiwan

  • TaiwanStockInfo : More than 1000 Taiwan stocks.
    Column stock_id, stock_name, kinds.

  • TaiwanStockPrice : All of Taiwan stock price of stockinfo. The date started in 1992.
    Column date, Trading_Volume, Trading_money, open, max, min, close, spread, Trading_turnover.

  • TaiwanStockFinancialStatements : All of Taiwan Stock Financial Statements of stockinfo.
    Column REV(Gross Revenue), COST(Cost of Goods Sold or Manufacturing), PRO(Gross Profit), GM(Gross margin), OE(Operating Expenses), OI(Operating Income), OIM(Operating Profit Margin), BTAX(Income before Tax), BTAXM(Income before Tax Margin), TAX(Income Tax Expense), NI(Net Income), NIM(Net Profit Margin), EPS(Earnings Per Share), stock_id, year, quar, date.

  • TaiwanStockStockDividend : Containing all of Taiwan Stock Stock Dividend of stockinfo.
    Column date, stock_id, Retained_Earnings, Capital_Reserve, Ex_right_trading_day, total_employee_bonus_stock_shares, Cash_dividend, Ex_dividend_transaction_day, total_employee_bonus_shares, Directors_remuneration.

  • InstitutionalInvestors : Institutional Investors buy or sell. ( Dealer, Dealer Hedging, Foreign Investor, Investment Trust )
    Column : name, buy, sell, date.

US

  • USStockInfo : More than 6200 US stocks.
    Column stock_id, name, Country, IPOYear, Country, MarketCap, Subsector.

  • USStockPrice : All of Taiwan stock price of stockinfo. The date started in 1992.
    Column date, Trading_Volume, Trading_money, open, max, min, close, spread, Trading_turnover.

Japan

  • JapanStockInfo : More than 2900 Japan stocks.
    Column stock_id, stock_name, Sector, Exchange.

  • JapanStockPrice : All of Taiwan stock price of stockinfo. The date started in 1992.
    Column date, Trading_Volume, Trading_money, open, max, min, close, spread, Trading_turnover.

International Financial Data

  • CrudeOilPrices : Brent or WTI.
    Column : price, date, name.

  • ExchangeRate : Exchange Rate. US dollar as the main currency. Contain CAD, CNY, EUR, GBP, JPY, TWD.
    Column : InterbankRate, InverseInterbankRate, date, country.

  • InterestRate : Interest Rate. Contain BCB, BOC, BOE, BOJ, CBR, ECB, FED, PBOC, RBA, RBI, RBNZ, SNB. The full name is column full_country_name.
    Column : date, interest_rate, country, full_country_name.

  • GovernmentBonds : Government Bonds. Contain Canada, China, France, Germany, Japan, Russia, Uk, Usa, Italy. Minimum 1-week(Russia), Maximum 50-Year(United Kingdom).
    Column : date, Price, Open, High, Low, ChangePercent, bound_type.

  • EnergyFuturesPrices : Energy Futures Prices. Contain Brent Oil Futures, Carbon Emissions Futures, Crude Oil WTI Futures, Gasoline RBOB Futures, Heating Oil Futures, London Gas Oil Futures, Natural Gas Futures.
    Column : date, Price, Open, High, Low, Vol, ChangePercent, futures_type.

  • GoldPrice : Gold Price. After 2018/7/20, the data is minutely data.
    Column : date, Price.
